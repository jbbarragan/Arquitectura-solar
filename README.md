# Solar·Arch — Análisis Bioclimático

This is a system that allows us to visualize advanced solar graphs and perform thermal comfort analysis based on EPW files. The system is currently hosted at https://arquitecturasolar.duckdns.org/
 and implements the innovation of analyzing temperatures across different dates of the year, allowing users to view a history of maximum and minimum values. This enables us to model optimal lighting and temperature optimizations, with the future goal of simulating intelligent structures.
