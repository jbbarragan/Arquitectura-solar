# Solar·Arch — Análisis Bioclimático

This is a system that allows us to visualize advanced solar graphs and perform thermal comfort analysis based on EPW files. The system is currently hosted at https://arquitecturasolar.duckdns.org/ and implements the innovation of analyzing temperatures across different dates of the year, allowing users to view a history of maximum and minimum values. This enables us to model optimal lighting and temperature optimizations, with the future goal of simulating intelligent structures.

## Overview

Solar·Arch is a web-based tool for **solar and bioclimatic analysis**.  
It integrates solar geometry visualization, 3D architectural model viewing, and climatic data analysis using **EPW weather files**.

The system generates visualizations and datasets that allow you to:

- Visualize the **3D solar chart** together with your architectural model
- Analyze **temperature data extracted from EPW files**
- Evaluate **thermal comfort conditions** using adaptive comfort models
- Explore **historical temperature ranges (min/max)** across different dates of the year

The platform uses **dry bulb temperature** as a reference and applies the **adaptive comfort temperature equation** to evaluate comfort conditions.

---

## Temperature Visualization

Based on the thermal comfort reference, the indicators change **color or temperature category every 2 to 2.5 °C**, allowing intuitive interpretation of thermal conditions.

The solar chart is generated in **both 2D and 3D**, using projections and a **bimonthly temperature averaging method** to generate color samples in the 3D diagram.

The resulting solar diagram is composed of:

- **6 × 24 panels**
- Representing **average hourly temperatures**
- For **each of the 12 months**

# Installation (AWS / Linux)

```bash
# 1. Clone or copy the project
cd /home/ubuntu/solar_arch

# 2. Create virtual environment
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Create required folders
mkdir -p uploads static

# 5. Start in production
bash start.sh 5000


To expose the service behind nginx, configure a proxy pass to:
http://127.0.0.1:5000

Usage
Solar Diagram

Open the application in your browser.

Click anywhere on the map or search for an address.

The polar solar diagram is automatically generated for that latitude.

3D Architectural Model

Drag a .obj, .mtl, or .stl file into the viewer, or use the 3D Model button.

The model is automatically positioned on top of the base plane/table.

Adjust the north orientation using the slider or the +90° / −90° buttons.

Use quick views: Top, Front, Iso.

EPW Weather File

Click Upload EPW and select an .epw file.

The system automatically generates:

Temperature heatmap (hour × month)

Thermal comfort table by month (cold / neutral / warm)

3D solar sphere colored by comfort

Download generated images and CSV files using the buttons in the lower toolbar.
